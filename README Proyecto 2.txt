La interfaz gráfica se puede separar en dos vertientes. 

La primera y principal posee tres botones: Jugar, Buscar y Administrar Base de Conocimientos. Se optó por marcar una diferencia pues muchas de las funcionalidades solicitadas se centraban únicamente en el manejo directo de la base de datos, por consiguiente, se consideró más eficiente que todo lo relacionado con el reescribir, leer, buscar y modificar las bases de datos fuesen un apartado al que se le accede con el botón de Administrar Base de Conocimientos en el menú inicial. Cabe destacar, que el programa viene cargado por defecto con la base de datos proporcionada por el enunciado, pero el usuario puede seleccionar otra en esa sección de administración. 

Con respecto a la representación gráfica del árbol, se tomó en cuenta la estructura vista en clase. Como esta es más direccionada a datos cortos (que no se superpongan unas hojas a otras), se dejó también la representación gráfica como una lista indentada multinivel. Dos maneras de ver gráficamente el árbol creado. 

La Tabla Hash, se carga, se llena y se evalúa cada vez que se le dé al botón de Buscar Animal. Cada vez que se inicie una búsqueda, se recorre el árbol por si hay algún nodo nuevo y, es únicamente la primera vez, se se lleva a acabo la función de insertar para todos los animales del árbol.
